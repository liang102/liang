选择器: 帮助你精准的选中想要的元素

简单选择器

ID选择器

-id对应的值: #id值{}

#str{
color: red;
}

<p id="str">Lorem ipsum dolor sit amet.</p>

元素选择器

  • 元素: 元素{}
p {
color: blue;
}


<p>Lorem ipsum dolor sit amet.</p>

类选择器

  • 类对应的值: .class{}
.left {
  background: coral;
  color: #fff;
}



<div class="left">Lorem.</div>

通配符选择器

  • *: 表示选中所有元素 *{}
*{
    color: red;
}

<div>Lorem.</div>
<p>Lorem.</p>
<h1>Lorem.</h1>

属性选择器

  • 匹配元素的属性: [属性=""]{}
[href="https://www.baidu.com"]{
    color: green;
}

<a href="https://www.baidu.com">百度</a>

伪类选择器: 选中某些元素的某种状态

  • 必须要按照顺序进行排列。
  • 爱恨法则：love hate
    • link: 超链接未访问时的状态。
    • visited： 超链接访问过后的状态。
    • hover： 鼠标悬停状态。
    • active: 激活状态，鼠标按下状态。
a:link{
    color: cadetblue;
}

a:visited{
    color: fuchsia;
}

a:hover{
    color: red;
}

a:active{
    color: chocolate;
}


# a 表示要给a元素添加

伪元素选择器: 可以生成子元素

  • before
  • after
span::before{   /* 选中span元素会生成一个子元素，可以理解为生成一个before元素 */
    content: "《";  /*  子元素中的内容  */
    color: red;     /*   颜色  */
}
span::after{
    content: "》";
    color: red;
}

<p>
    快来和我一起学习<span>HTML和CSS</span>吧!
</p>
